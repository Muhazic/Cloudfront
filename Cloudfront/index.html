Using Terraform, I established a resilient infrastructure with Route 53 routing to an Application Load Balancer, directing traffic to a private instance hosting a dynamic website and linked to an S3 bucket. AWS IAM ensured secure resource management, while CloudWatch monitored S3 operations with SNS alerts. A bastion host in the public subnet enabled secure SSH access, and agent forwarding allowed key passage. The multi-region, multi-AZ design, route tables, NAT gateways, and NACLs ensured robustness. This project highlights my Terraform expertise, emphasis on security, and architecture's resilience.

This Java program offers an interactive experience for playing Tower of Hanoi. The code prompts the user to specify the number of disks to be used in the game and calculates the minimum number of moves required to solve the puzzle efficiently. It employs an array of stack data structures to represent the three towers, enforcing the rules of the game and allowing the user to move disks between towers. After each move, the program visually displays the current state of the towers, illustrating the arrangement of disks and the number of moves made. The game continues until the user successfully transfers all disks to a different tower, at which point it announces the total number of moves taken to accomplish the task.
